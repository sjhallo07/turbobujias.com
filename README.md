# TurboBujías Pro

Premium automotive e-commerce platform featuring AI-powered diagnostics, document-based technical manual learning, and RAG-integrated smart support.

## Features

- **AI-Powered Diagnostics:** Get smart recommendations and diagnostic assistance.
- **Automotive E-commerce:** Browse and purchase high-performance spark plugs and automotive parts.
- **Technical Manual Learning:** Intelligent RAG (Retrieval-Augmented Generation) system to query technical manuals.
- **Smart Support:** AI-driven chatbot for customer support.
- **Administrative Dashboard:** Manage products, orders, and data.

## Technologies Used

- **Frontend:** React, TypeScript, Tailwind CSS, Motion (Animations)
- **Backend:** Node.js, Express, Vite
- **Database:** Firebase (Firestore)
- **AI/LLM:** LangChain, OpenAI, Google Generative AI
- **Integrations:** Google Maps, HTML5 QR Code, XLSX/CSV Parsing for bulk data

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm (v10+)

### Setup

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Set up environment variables (copy `.env.example` to `.env` and fill in necessary keys).

4. Run the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:3000`.

## Build for Production

```bash
npm run build
npm start
```
